Problem 5: Greater Number
Time Limit: 1 Seconds
Memory Limit: 2.93 MB

Statement
You will be given two integers A and B. Print the greater number.
If both numbers are equal, print "Equal".

Constraints
-1000 <= A, B <= 1000

Input format
The input consists of two integers A and B.

Output Format
Print the greater number or Equal if both are same.

Sample Input 1
10 20
Sample Output 1
20
