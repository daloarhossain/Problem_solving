Problem 12: M. Capital or Small or Digit
time limit per test : 1 second
memory limit per test : 256 megabytes

Statement
Given a letter X. Determine whether X is a Digit or an Alphabet, and if it is an Alphabet, determine whether it is Capital Case or Small Case.

Note:
Digits in ASCII: '0' = 48, '1' = 49, ... etc
Capital letters in ASCII: 'A' = 65, 'B' = 66, ... etc
Small letters in ASCII: 'a' = 97, 'b' = 98, ... etc

Input
Only one line containing a character X which will be a capital or small letter or digit.

Output
Print a single line containing "IS DIGIT" if X is a digit.
Otherwise, print "ALPHA" in the first line followed by a new line containing "IS CAPITAL" if X is a capital letter or "IS SMALL" if X is a small letter.

Examples
Input
A
Your output should be:
ALPHA
IS CAPITAL

Input
9
Your output should be:
IS DIGIT

Input
a
Your output should be:
ALPHA
IS SMALL
