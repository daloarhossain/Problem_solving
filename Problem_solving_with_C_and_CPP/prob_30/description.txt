**Problem 30: Reverse and Odd**
time limit per test: 1 second
memory limit per test: 256 megabytes

You will be given a positive integer N and an array A of size N.
Suppose the index starts from 0 — you need to print all the values at **odd indexes** in **reverse order**.

**Example:**
If the input is:
6
10 20 30 40 50 60
You need to print **60 40 20** as their indexes are 5, 3, and 1 respectively.

**Input**
First line will contain an integer N.
Second line will contain N integers representing the array A.

**Constraints**
1 ≤ N ≤ 10⁵
1 ≤ A[i] ≤ 1000

**Output**
Print all the values of odd indexes in reverse order, separated by a space.

**Examples**

Input
5
10 20 30 40 50

Output
40 20

Input
6
0 1 2 3 4 5

Output
5 3 1
