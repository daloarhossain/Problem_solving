Problem 36: Pattern 3

**Problem Statement**
You're given a number **N** as input. You need to print a pattern using capital English alphabets.
For each line **i** (starting from 1), print the corresponding alphabet (starting from 'A') **i** times.

For example,
If **N = 6**, then the output will look like the following example.

---

**Input Format**
Only one line containing a number **N**.

---

**Output Format**
Print the pattern as shown in the example below.

---

**Constraints**
1 ≤ N ≤ 10

---

**Sample Input 0**
6

**Sample Output 0**
A
B B
C C C
D D D D
E E E E E
F F F F F F
