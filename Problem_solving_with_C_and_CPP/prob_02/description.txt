Problem 2: Even or Odd
Time Limit: 1 Seconds
Memory Limit: 2.93 MB

Statement
In this problem, you will be given an integer N. Print "Even" if the number is divisible by 2; otherwise, print "Odd".

Constraints
-1000 <= N <= 1000

Input format
The input consists of an integer N.

Output Format
Print Even if divisible by 2, otherwise Odd.

Sample Input 1
8
Sample Output 1
Even
