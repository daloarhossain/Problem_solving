Problem 24: Sum of Consecutive Odd Numbers
time limit per test: 1 second
memory limit per test: 256 megabytes

Given two numbers X and Y. Print the sum of all odd numbers between them, excluding X and Y.

Input
First line contains a number T (1 ≤ T ≤ 10) — the number of test cases.
Next T lines will contain two numbers X and Y (0 ≤ X, Y ≤ 10⁴).

Output
Print the sum of all odd numbers between X and Y (excluding X and Y).

Examples

Input
3
5 6
10 4
4 9

Output
0
21
12
