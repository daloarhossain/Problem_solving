Problem 6: Simple Calculator
Time Limit: 1 Seconds
Memory Limit: 2.93 MB

Statement
You will be given two integers A, B and an operator op (+, -, *, /). Perform the operation and print the result.

Constraints
1 <= A, B <= 1000

Input format
Input contains two integers and one operator separated by space.

Output Format
Print the result of the operation.

Sample Input 1
10 + 5
Sample Output 1
15
