Problem 01: Zero or Non Zero
Time Limit: 1 Seconds
Memory Limit: 2.93 MB

Statement
In this problem, you will be given an integer number N. Print "Zero" if the number equals 0 and "Non Zero" otherwise.

Constraints
-1000 <= N <= 1000

Input format
The input consists of an integer N.

Output Format
Print Zero if the number equals 0 and Non Zero otherwise.

Sample Input 1:
5
Sample Output 1:
Non Zero
